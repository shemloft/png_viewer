# Декодировщик PNG файлов

Автор: Шемякина Дарья 

## Описание
Данное приложение является декодировщиком PNG файлов с возможностью просмотра изображения и внутренней структуры файла.

## Требования
* PyQt5 для графической версии

## Состав
* Графическая и консольная версия: `png.py`
* Модули: `png_viewer/`
* Тесты: `tests/`
* Дополнительные скрипты: `check_correctness.py`, `get_test_pics.py`
* Ссылка на набор тестовых изображений: http://schaik.com/pngsuite/PngSuite-2017jul19.zip

## Графическая и консольная версия
Справка по запуску: `./png.py --help`
Пример запуска (графическая версия): `./png.py –v file_name.png`
Пример запуска (консольная версия): `./png.py file_name.png –с 1`
Пример запуска (обе версии): `./png.py –с -v file_name.png`

## Справка по консольной версии
Вслед за ключом –c могут идти следующие опции:
0: вывод полей заголовка без описания
1: вывод полей заголовка с описанием
2: вывод полей заголовка с описанием и информации о цветах пикселей

## Справка по дополнительным скриптам
Скрипт для скачивания и распаковки тестовых изображений.
Пример запуска: `./get_test_pics.py http://schaik.com/pngsuite/PngSuite-2017jul19.zip`

Скрипт для запуска программы на наборе тестовых изображений с подсчетом правильно изображенных.
Пример запуска: `./check_correctness.py png_test_pics "python png.py –c 1”`

## Подробности реализации
Утилита отображает простые PNG изображения, используя только критические чанки. Чересстрочные (interlaced) изображения не поддерживаются.
